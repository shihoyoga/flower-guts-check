# Flower Guts Check - 腸内環境診断

「Flower Guts Check」は、簡単な質問に答えるだけであなたの腸内環境を診断し、お花畑に例えて結果を表示するウェブアプリケーションです。

## 特徴

- **簡単診断**: 食生活、生活習慣、体調に関する質問に答えるだけ。
- **視覚的な結果**: 腸内環境の状態を「砂漠」から「楽園」までの5段階で表現。
- **アドバイス機能**: 診断結果に基づいた具体的な改善アクションを提案。
- **レスポンシブデザイン**: スマートフォンでも快適に利用可能。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「診断をはじめる」ボタンをクリックします。
3. 全ての質問に回答します。
4. 診断結果とアドバイスを確認します。
